    data policy

    We store your data (uploaded files) in your device (browser’s indexedDB) encoded in base64 format. For backup and multi-device support purpose, we have to keep a copy of it in our server side storage which we guarantee to be secure.

    Other than that, we don’t use it for finetuning the LLM used for the Autofill. User data is sent to the server side everytime you request to Autofill.

    Data Policy is subjected to change and changes will be notified to you via email.
